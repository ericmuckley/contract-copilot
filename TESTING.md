# Testing

To run tests, first configure the repo according to the instructions in `README.md`. Then run

```bash

npm run test

```

The tests will run for the following chatbot tool-calling functions:

- GetProjectDetailsTool
- GetContractDetailsTool
- GetContractEditsSummaryTool
- AddNoteToContractTool
- UpdateProjectTasksTool
- CreateNewContractVersionTool
- CreateNewContractTool
